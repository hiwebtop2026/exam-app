# exam-app - 系统集成项目管理工程师备考系统

## 📖 项目简介

这是一个专为系统集成项目管理工程师考试设计的备考系统，包含学习计划、学习重点、学习笔记、真题练习、冲刺复习、模拟考试等完整功能。

## 🎯 功能特性

### 1. 📅 学习计划
- 自动生成8周学习计划
- 进度跟踪与可视化
- 每日任务管理

### 2. 📚 学习重点
- 17章完整知识点
- 重要程度标注
- 核心公式速查

### 3. 📝 学习笔记
- 自由创建笔记
- 自动保存功能
- 分类管理

### 4. ✍️ 真题练习
- 历年真题（2020-2024）
- 分类筛选
- 练习统计

### 5. 🚀 冲刺复习
- 复习卡片（翻转记忆）
- 核心公式速记
- 重点知识快速回顾

### 6. 📊 模拟考试
- 全真模拟环境
- 计时功能
- 成绩分析
- 知识点薄弱环节识别

### 7. ❌ 错题本
- 自动记录错题
- 详细解析
- 针对性复习

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 找到项目文件夹
2. 双击 `index.html` 文件
3. 在浏览器中自动打开应用

### 方式二：使用本地服务器

```bash
# 进入项目目录
cd exam-prep-app

# 使用Python启动服务器
python -m http.server 8080

# 或使用Node.js
npx http-server

# 打开浏览器访问
# http://localhost:8080
```

## 💾 数据存储

应用使用 **IndexedDB** 进行本地数据存储，所有数据保存在浏览器中：
- 学习进度
- 学习笔记
- 考试记录
- 错题本
- 练习统计

**注意**：清除浏览器数据会导致学习记录丢失，请注意备份。

## 📝 使用说明

### 学习计划

1. 点击左侧菜单"学习计划"
2. 查看自动生成的8周计划
3. 勾选已完成的任务
4. 系统自动计算进度

### 模拟考试

1. 点击左侧菜单"模拟考试"
2. 选择"上午考试"或"下午考试"
3. 开始答题（注意计时）
4. 提交后查看成绩和分析

### 冲刺复习

1. 点击"冲刺复习"
2. 使用翻转卡片记忆知识点
3. 点击卡片翻转查看答案
4. 标记"认识"或"不认识"

### 错题本

- 做错的题目自动进入错题本
- 可以查看详细解析
- 支持重新练习和移除

## 🔧 自定义扩展

### 添加新题目

编辑 `src/data/questions.js` 文件，按以下格式添加：

```javascript
{
    id: 'q2025001',
    year: 2025,
    type: 'morning',  // morning 或 afternoon
    chapter: 'project_integration',
    chapterName: '项目整体管理',
    title: '题目内容',
    question: '完整题目描述',
    options: [
        { label: 'A', text: '选项A' },
        { label: 'B', text: '选项B' },
        { label: 'C', text: '选项C' },
        { label: 'D', text: '选项D' }
    ],
    answer: 'C',
    explanation: '详细解析',
    tags: ['标签1', '标签2'],
    difficulty: 'medium'  // low, medium, high
}
```

### 添加新章节

编辑 `src/data/chapters.js` 文件。

### 添加新公式

编辑 `src/data/formulas.js` 文件。
