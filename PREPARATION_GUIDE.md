## 核心公式
### 进度管理公式
- EF = ES + DU（最早完成时间）
- LS = LF - DU（最晚开始时间）
- TF = LS - ES = LF - EF（总时差）
- TE = (O + 4M + P) / 6（PERT期望时间）
- sigma = (P - O) / 6（PERT标准差）

### 成本管理公式
- CV = EV - AC（成本偏差）
- SV = EV - PV（进度偏差）
- CPI = EV / AC（成本绩效指数）
- SPI = EV / PV（进度绩效指数）
- EAC = BAC / CPI（完工估算）
- ETC = (BAC - EV) / CPI（完工尚需估算）
